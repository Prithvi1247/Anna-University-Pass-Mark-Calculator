Pass Condition Logic
BOTH of these must be satisfied:

Condition 1: external_marks ≥ 32 // 40% of 80
Condition 2: (internal + external) ≥ 50 // 50% overall

required_for_condition2 = 50 - internal
required_to_pass = max(32, required_for_condition2)
requiredForGrade(internal, targetTotal):
  return max(32, targetTotal - internal)
